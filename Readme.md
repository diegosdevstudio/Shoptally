# 🛒 ShopTally

### *Smart Shopping List Manager*

**ShopTally** is a professional-grade shopping list tool designed for anyone who wants to shop smarter, track spending in real time, and share a clean receipt with a single tap. Part of the **DiegosDevStudio** suite, it delivers a luxury mobile-first experience with zero friction — no account, no install, no nonsense.

-----

## ✨ Key Features

- **Multi-List Management:** Create and manage unlimited named shopping lists, each with a custom icon and persistent storage across sessions.
- **Dual-Mode Workflow:** Switch between **Planning Mode** (build your list at home) and **At the Store Mode** (tap to add prices as you shop in real time).
- **Live Price Tracking:** Add per-item prices and quantities on the fly. Running totals, line totals, and “no price yet” alerts update instantly.
- **Receipt Generator:** Produce a clean, shareable receipt with itemized pricing, totals, and an optional personal note — copy it to any messaging app in one tap.
- **Persistent Storage:** All lists and items survive browser refreshes via `localStorage` — no account or backend required.
- **Signature UI:** A high-end “Midnight & Cyan” glassmorphism interface with animated grid and nebula backgrounds, designed for professional environments.

-----

## 🛠️ Technical Architecture

- **Core Stack:** Zero-dependency, single-file HTML/CSS/JS architecture for maximum speed and portability
- **Design System:** DiegosDevStudio “Cinematic Command Center” (Deep Charcoal `#121826` | Electric Cyan `#22D3EE` | Royal Blue `#2563EB`)
- **Typography:** Rajdhani (display) · Outfit (UI) · JetBrains Mono (mono) via Google Fonts
- **Security:** All data stays client-side via `localStorage` — nothing is ever transmitted
- **Performance:** No build step, no npm, no frameworks — open and run

-----

## 🚀 Quick Start

1. Clone or download the repository
1. Open `ShopTally.html` in any modern web browser
1. Tap **＋ CREATE NEW LIST** to create your first list
1. Add items in **Planning Mode**, then switch to **At the Store Mode** to add prices as you shop
1. Tap **GENERATE RECEIPT** to produce and copy a shareable summary
